GESTIONE BIBLIOTECA UNISA
Il progetto consiste nello sviluppo di un sistema software per la gestione di una biblioteca universitaria, che permette di organizzare in modo efficiente libri, utenti e prestiti, riducendo al minimo le operazioni manuali e gli errori.  
L’applicazione offre funzionalità di inserimento, modifica, ricerca ed eliminazione di libri e utenti, controllando l’unicità di ISBN e matricola/e‑mail e mantenendo l’archivio sempre aggiornato. 
Il sistema gestisce i prestiti applicando vincoli di dominio (numero finito di copie, massimo tre prestiti attivi per utente, evidenziazione dei ritardi) e salva automaticamente su file tutte le informazioni, che vengono ricaricate all’avvio per garantire continuità e affidabilità dei dati.
